## 简介

**养成好习惯，保持动力，共同成长。**

GrowNest 是您的个人成长伴侣，将严谨的计划与游戏化的乐趣相结合。无论您是争取更好成绩的学生、终身学习者，还是指导孩子教育的家长，GrowNest 都能为您提供成功所需的结构和动力。

**主要功能**：

**[智能规划与实时追踪]** 轻松创建自定义学习计划。借助全新的实时活动支持，您可以直接从锁屏或灵动岛追踪您的专注进度，不受任何干扰。

**[游戏化奖励系统]** 将努力转化为切实的奖励！完成任务 or 保持连胜即可获得积分。设置自定义的“奖励商店”（例如，“游戏 1 小时”）以保持高昂的动力。

**[家长与共同学习模式]** 面向家庭的独特功能。家长可以切换到管理员模式以查看进度、分配任务和批准奖励。营造健康、透明和充满鼓励的环境。

**[高级学习工具]**

* 语音录制：在学习期间记录想法或讲座。

* iCloud 同步：确保您的数据在所有设备上安全同步。

* 阅读管理：记录页数并管理您的“待读”和“已读”书架。

**[可视化您的成长]**

加入“7天早起”等挑战，看着您的努力转化为美丽的数据趋势和成就徽章。
