## 説明

**より良い習慣を築き、モチベーションを維持し、共に成長しましょう。**

GrowNest は、厳格な計画とゲーミフィケーションの楽しさを組み合わせた、あなたの個人的な成長のパートナーです。成績アップを目指す学生、生涯学習者、または子供の教育を導く親であっても、GrowNest は成功に必要な構造とモチベーションを提供します。

**主な機能**:

**[スマートプランニング & ライブトラッキング]** カスタム学習スケジュールを簡単に作成できます。新しいライブアクティビティのサポートにより、ロック画面や Dynamic Island から直接、集中力の進行状況を追跡でき、気を散らすことがありません。

**[ゲーム化された報酬システム]** 努力を具体的な報酬に変えましょう！タスクを完了したり、連続記録を維持したりしてポイントを獲得します。モチベーションを高く保つために、カスタムの「報酬ストア」（例：「1時間ゲーム」）を設定します。

**[親と共同学習モード]** 家族向けのユニークな機能です。親は管理者モードに切り替えて、進捗状況の確認、タスクの割り当て、報酬の承認を行うことができます。健全で透明性があり、励まし合える環境を育みます。

**[高度な学習ツール]**

* 音声録音：学習セッション中に考えや講義を記録します。

* iCloud 同期：データを安全に保ち、すべてのデバイス間で同期します。

* 読書管理：ページを記録し、「読みたい本」と「読み終わった本」の本棚を管理します。

**[成長の可視化]**

「7日間早起き」のようなチャレンジに参加し、あなたの努力が美しいデータトレンドや達成バッジに変わるのを見てみましょう。
