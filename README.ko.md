## 설명

**더 나은 습관을 기르고, 동기를 부여받으며, 함께 성장하세요.**

GrowNest는 철저한 계획과 게이미피케이션의 재미를 결합한 개인 성장 동반자입니다. 성적 향상을 위해 노력하는 학생이든, 평생 학습자이든, 자녀 교육을 지도하는 부모이든, GrowNest는 성공에 필요한 구조와 동기를 제공합니다.

**주요 기능**:

**[스마트 계획 및 실시간 추적]** 맞춤형 학습 일정을 손쉽게 만드세요. 새로운 실시간 현황(Live Activities) 지원을 통해 잠금 화면이나 Dynamic Island에서 방해 없이 집중 진행 상황을 직접 추적할 수 있습니다.

**[게임화된 보상 시스템]** 노력을 확실한 보상으로 바꾸세요! 작업을 완료하거나 연속 기록을 유지하여 포인트를 획득하세요. 맞춤형 "보상 상점"(예: "1시간 게임")을 설정하여 높은 동기를 유지하세요.

**[학부모 및 공동 학습 모드]** 가족을 위한 독특한 기능입니다. 부모는 관리자 모드로 전환하여 진행 상황을 확인하고, 작업을 할당하고, 보상을 승인할 수 있습니다. 건강하고 투명하며 격려하는 환경을 조성하세요.

**[고급 학습 도구]**

* 오디오 녹음: 학습 세션 중에 생각이나 강의를 기록하세요.

* iCloud 동기화: 모든 기기에서 데이터를 안전하게 동기화된 상태로 유지하세요.

* 독서 관리: 읽은 페이지를 기록하고 "읽을 책" 및 "다 읽은 책" 서재를 관리하세요.

**[성장 시각화]**

"7일 일찍 일어나기"와 같은 챌린지에 참여하고 여러분의 노력이 아름다운 데이터 트렌드와 성취 배지로 변하는 것을 지켜보세요.
