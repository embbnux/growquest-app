## 簡介

**養成好習慣，保持動力，共同成長。**

GrowNest 是您的個人成長伴侶，將嚴謹的計畫與遊戲化的樂趣相結合。無論您是爭取更好成績的學生、終身學習者，還是指導孩子教育的家長，GrowNest 都能為您提供成功所需的結構和動力。

**主要功能**：

**[智慧規劃與即時追蹤]** 輕鬆建立自訂學習計畫。借助全新的即時動態支援，您可以直接從鎖定畫面或動態島追蹤您的專注進度，不受任何干擾。

**[遊戲化獎勵系統]** 將努力轉化為切實的獎勵！完成任務或保持連勝即可獲得積分。設定自訂的「獎勵商店」（例如，「遊戲 1 小時」）以保持高昂的動力。

**[家長與共同學習模式]** 物向家庭的獨特功能。家長可以切換到管理員模式以查看進度、分配任務和批准獎勵。營造健康、透明和充滿鼓勵的環境。

**[進階學習工具]**

* 語音錄製：在學習期間記錄想法或講座。

* iCloud 同步：確保您的資料在所有裝置上安全同步。

* 閱讀管理：記錄頁數並管理您的「待讀」和「已讀」書架。

**[視覺化您的成長]**

加入「7天早起」等挑戰，看著您的努力轉化為美麗的數據趨勢和成就徽章。
